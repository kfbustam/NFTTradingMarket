=============

Backend:

Go to the directory "NFTTradingMarket/initial" and run the following command to start the server:

"./mvnw clean spring-boot:run"


=============

Frontend:

Go to the directory "NFTTradingMarket/frontend" and run the following command to install all the packages:

"npm install" and/or "yarn install"

then start the frontend react server by running in that same directory:

"npm run start"

==============

Access the app at http://localhost:3000/
